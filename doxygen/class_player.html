Detailed Description

Player class.

Player is controlled by the user using WASD keys. The player can fire lasers to kill Aliens as well as activating smart bombs and hyperjumps. Smart bombs kill everything on screen but takes a minute to recharge. The hyperjump moves the player to a random location.
